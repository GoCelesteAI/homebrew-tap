# homebrew-tap

Homebrew tap for [GoCelesteAI](https://github.com/GoCelesteAI) tools.

## Usage

```sh
brew install gocelesteai/tap/cargoterm
```

Or tap first, then install:

```sh
brew tap gocelesteai/tap
brew install cargoterm
```

## Available formulae

| Formula | Description |
| --- | --- |
| [`cargoterm`](./Formula/cargoterm.rb) | AI-augmented terminal in Rust — a local LLM translates natural language into shell commands. Repo: [GoCelesteAI/cargoterm](https://github.com/GoCelesteAI/cargoterm) |
